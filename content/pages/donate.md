# Пожертвовать

АНО «Линка» принимает добровольные пожертвования на осуществление уставной деятельности, направленной на развитие доступной среды и повышение коммуникативных навыков людей с ОВЗ.

Пожертвование является добровольным. Оно не является оплатой товара, работы, услуги, курса, консультации, программного обеспечения или цифрового сервиса.

Средства пожертвований используются для достижения уставных целей организации: развития и поддержки программ альтернативной и дополнительной коммуникации, подготовки материалов, сопровождения пользователей и организационной работы АНО «Линка».

## Прием платежей

Для пожертвования укажите сумму, периодичность, ФИО и email. Эти данные нужны для обработки платежа, уведомлений и исполнения требований законодательства.

Доступны разовые и ежемесячные пожертвования. При ежемесячном пожертвовании YooKassa сохраняет способ оплаты для последующих списаний выбранной суммы один раз в месяц.

[linka_donation_form]

Нажимая кнопку «Пожертвовать», жертвователь подтверждает, что ознакомился с публичной офертой о добровольном пожертвовании и политикой обработки персональных данных.

Для разовых пожертвований YooKassa показывает все доступные для магазина способы оплаты. Для ежемесячных пожертвований доступны способы оплаты, которые поддерживают сохранение способа оплаты и последующие списания. Отключить ежемесячное пожертвование и удалить сохраненный способ оплаты можно самостоятельно по защищенной ссылке управления из письма.
