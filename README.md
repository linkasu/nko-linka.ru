# nkolinka.ru

Официальный сайт АНО "Линка" на WordPress для домена `nkolinka.ru`.

## Назначение

Сайт должен заменить старый `linka.su` как главный публичный сайт проекта Linka и юридического лица АНО "Линка".

## Архитектура

- WordPress в Yandex Cloud Serverless Containers.
- Docker image собирается через GitHub Actions.
- MariaDB/MySQL работает в Docker на сервере `37.230.192.57`.
- Медиа WordPress хранятся в Yandex Object Storage.
- HTTPS через Yandex Certificate Manager.
- Runtime secrets хранятся вне репозитория.

## Пожертвования

На сайте разрешен только раздел добровольных пожертвований на уставную деятельность АНО "Линка". Оплата товаров, услуг, курсов, консультаций и программного обеспечения через сайт не предусматривается.

После одобрения ЮKassa активная форма пожертвования включается только через runtime secrets. Ключи YooKassa не хранятся в репозитории. Добровольные пожертвования принимаются без кассовых чеков на товары, работы или услуги.

Ежемесячные пожертвования подготовлены в коде, но включаются только после подключения автоплатежей YooKassa для магазина и установки runtime `YOOKASSA_RECURRING_ENABLED=true`. Для ежемесячных пожертвований предусмотрено самостоятельное отключение по защищенной ссылке из email с удалением сохраненного способа оплаты из системы сайта.

Успешные платежи учитываются в локальных таблицах WordPress для публичной общей суммы и read-only страницы «Пожертвования» в админке. После успешного разового платежа отправляется благодарственное письмо. Ежемесячные списания запускает YC timer `nko-linka-recurring-donations` каждые 15 минут через IAM-защищенный внутренний endpoint приватного Serverless Container; публичный API Gateway блокирует весь `/internal/*`.
